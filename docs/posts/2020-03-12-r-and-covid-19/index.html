Data

For data we will use the coronavirus dataset, which is a conveniently tidied table based on the Johns Hopkins University Center for Systems Science and Engineering (JHU CCSE). The package is published to CRAN, but it is updated on GitHub on a daily basis. To get the latest version, we run:


remotes::install_github("RamiKrispin/coronavirus")

We can have a look at the data. As we see, each row is a particular record with the number of cases, confirmed, dead or recovered, for a given region. Geographical coordinates are added for map plots.


library(coronavirus)
# View the data

paged_table(coronavirus)

Each row of this table gives the number of cases reported per each day and region between 2020-01-22 and 2020-03-13.
